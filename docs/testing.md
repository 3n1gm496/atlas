# Testing

## Suite disponibili
- Unit test (Vitest): validazioni e utility.
- E2E smoke (Playwright): caricamento homepage.

## Comandi
- `npm run typecheck`
- `npm run test`
- `npm run build`

## Estensioni raccomandate
- Integration test su route handlers (`/api/entries`, `/api/submit`, `/api/analytics/overview`).
- E2E completo per workflow contributor -> editor review -> publish.
